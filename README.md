##  Project Overview

This project is a UI/UX design case study for a specialized digital marketplace dedicated to buying and selling second-hand baby and kids’ products in Egypt.
The idea is based on strong market validation and research showing a high volume of unused baby items and a growing demand for affordable alternatives. Currently, buying and selling occurs through scattered Facebook groups and platforms such as Dubizzle, which lack structure, trust systems, and optimized user experience.
Our team aims to design both a responsive website and a mobile application that provide a safe, organized, and user-friendly experience tailored specifically for mothers and families.
The project will include user research, problem definition, UX strategy, wireframes, high-fidelity designs, and interactive prototypes.

## Project Plan (5 Weeks)

###  Week 1 — [Research & Problem Definition]
- Conduct user research and analyze market validation data.
- Identify user pain points from Facebook groups and existing platforms.
- Define problem statement and value proposition. 
### Week 2 — [UX Strategy & Information Architecture]
- Create user personas.
- Develop user journeys and task flows.
- Design information architecture for website and mobile app.  
### Week 3 — [Wireframing]
- Create low-fidelity wireframes.
- Review and iterate based on feedback.
- Validate user flows and usability logic.
### Week 4 — [UI Design & Design System]
- Develop branding direction (colors, typography, UI components).
- Create high-fidelity screens for mobile app.
- Design responsive website layouts.
### Week 5 — [Prototyping & Presentation]
- Build interactive prototypes.
- Conduct basic usability testing.
- Refine final designs.
- Prepare final case study presentation
